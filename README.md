## 📋 Description

An automated bot for the Knidos Testnet that handles:
- ✅ Daily Check-in
- 🎮 Auto Play Game (game_1)
- 🔄 Multi-account support
- 🌐 Proxy support
- ⏱️ Auto cycle every 24 hours

---

## ⚙️ Requirements

- Python 3.8+
- pip packages (see Installation)

---

## 📦 Installation

```bash
git clone https://github.com/febriyan9346/KNIDOS-BOT.git
cd KNIDOS-BOT
pip install -r requirements.txt
```

---

## 🗂️ Setup

1. **accounts.txt** – Add your private keys (one per line):
   ```
   your_private_key_here
   another_private_key
   ```

2. **proxy.txt** *(optional)* – Add proxies (one per line):
   ```
   http://user:pass@ip:port
   http://ip:port
   ```

---

## ▶️ Usage

```bash
python bot.py
```

Select mode:
- `1` → Run with proxy
- `2` → Run without proxy

---

## 📌 Features

| Feature | Description |
|---|---|
| Auto Login | Signs in using wallet private key |
| Daily Check-in | Claims daily check-in reward automatically |
| Auto Play Game | Plays `game_1` and claims points |
| Multi-Account | Processes all accounts in `accounts.txt` |
| Proxy Support | Rotates proxies per account |
| Auto Cycle | Repeats every 24 hours |
